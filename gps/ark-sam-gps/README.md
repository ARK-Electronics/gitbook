# ARK SAM GPS

### Hardware Setup <a href="#hardware-setup" id="hardware-setup"></a>

#### Wiring <a href="#wiring" id="wiring"></a>

The ARK SAM GPS is connected to a UART/I2C port a Pixhawk standard 6 pin JST GH cable.
