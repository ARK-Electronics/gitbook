# PX4 Instructions

## PX4 Setup

Enable [SENS\_EN\_ADIS165X](https://docs.px4.io/main/en/advanced_config/parameter_reference.html#SENS_EN_ADIS165X)

The[ adis16507 driver](https://github.com/PX4/PX4-Autopilot/blob/main/boards/ark/fmu-v6x/default.px4board#L20) must be complied in the board firmware for the parameter to appear.
